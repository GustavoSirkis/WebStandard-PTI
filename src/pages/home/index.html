Tecnologias

Angular by GooGle

framework JavaScript mantido pelo Google, utilizado para criar aplicativos web dinâmicos e SPA. O Angular oferece uma estrutura completa com recursos como injeção de dependências, gerenciamento de estado e roteamento.

React by Meta

biblioteca JavaScript de código aberto desenvolvida pelo Facebook, usada para construir interfaces de usuário reativas e componentizadas. O React permite a criação de aplicativos de página única (SPA) e promove a reutilização de componentes.

JQuery by OpenJS

Biblioteca JavaScript que simplifica a manipulação do DOM, animações, requisições AJAX e outras tarefas comuns. Embora tenha sido muito popular, seu uso tem diminuído com a ascensão de frameworks como React e Vue.js.

Gerenciamento de Versões

Um sistema de versionamento permite que várias pessoas trabalhem no mesmo conjunto de arquivos (repositório) ao mesmo tempo evitando conflitos entre as alterações das versões do projeto. Cada membro do time de desenvolvimento tem sua “cópia” dos arquivos que ao final das alterações é colocada junto das versões alteradas dos demais.
